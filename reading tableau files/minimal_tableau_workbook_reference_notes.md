# Minimal Tableau TWB reference

This is a short teaching / inspection version of the workbook XML.

It keeps one example each of:
- top-level workbook sections
- a Parameters datasource
- a main datasource with field definitions and one calc
- a worksheet with datasource-dependencies, style, panes, rows, and cols
- a dashboard with zones

It removes most workbook-specific logic, repeated fields, and repeated formatting so the file stays small and readable.

Use it as a structural map, not as a full-fidelity production workbook.
